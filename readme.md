# Reminder app showing notification at {x}th mins of every 10 mins

## Install the packages.
```
go mod tidy
```

## Run the application

```
go run main.go 1
```
It will show the notification 1st mins of every 10mins


## Build the application

```
go build main.go
```

## Run applicatin in background
```
nohup ./main &
```
